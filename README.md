# Taller 28 / 11 / 2025 - Sistema de Login y Gestión de Equipos de Fútbol (Python)

Este repositorio contiene un pequeño sistema en Python que utiliza funciones, archivos CSV y un orquestador para controlar el flujo de la aplicación. El objetivo es aplicar todos los conceptos vistos en clase de una manera práctica y organizada.

1. Lee un archivo CSV de usuarios.
2. Contiene un sistema de login que valida el usuario y la contraseña.
3. Si el login es exitoso, permite el acceso a un CRUD de equipos de fútbol colombianos.
4. Si el usuario no existe o la contraseña es incorrecta, muestra un mensaje de error y tras 3 intentos fallidos, finaliza la ejecución.
5. El código está estructurado en módulos separados para fomentar buenas prácticas.
